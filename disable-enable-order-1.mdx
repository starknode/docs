---
title: "Toggle order status"
api: "PATCH /order/{order_id}/toggle"
description: "Enable or disable an order template by flipping the active flag."
authMethod: "key"
---

<ParamField header="x-api-key" type="string" required placeholder="your-api-key">
  Your StarkNode API key. You can find it in your [dashboard](https://app.starknode.io).
</ParamField>

<ParamField path="order_id" type="string" required>
  Order CUID.
</ParamField>

<RequestExample>

```bash cURL
curl --request PATCH \
  --url https://api.starknode.io/order/clxxxxxxxxxxxxxxxxxxxxxxxx/toggle \
  --header 'x-api-key: <api_key>'
```

```javascript Node.js
const orderId = "clxxxxxxxxxxxxxxxxxxxxxxxx";
const response = await fetch(
  `https://api.starknode.io/api/order/${orderId}/toggle`,
  { method: "PATCH", headers: { "x-api-key": "<api_key>" } }
);
const data = await response.json();
```

```python Python
import requests

order_id = "clxxxxxxxxxxxxxxxxxxxxxxxx"
response = requests.patch(
    f"https://api.starknode.io/api/order/{order_id}/toggle",
    headers={"x-api-key": "<api_key>"},
)
data = response.json()
```

</RequestExample>
